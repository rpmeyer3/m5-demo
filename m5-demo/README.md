Mobile app concept that scans grocery receipts to auto-build a pantry with expiration tracking and recipe suggestions.

visit at:
https://freshlykeptsecret.vercel.app/


Best viewed at iPhone 13/14 size (375x812) using devtools device toolbar or just on the webpage is perfectly fine

 Task flows

1. Check expiration - Inventory -> tap item.
2. Scan receipt - Home -> Scan Receipt -> Capture -> Confirm.
3. Connect store + notifications - Settings -> Link My Grocery Accounts -> Kroger -> Authorize.
